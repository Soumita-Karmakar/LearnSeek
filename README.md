# 📚 LearnSeek

LearnSeek is a full-stack web application that connects **students and teachers** on a single platform.
Students can search for teachers based on subject or location, send messages, and provide feedback, while teachers can respond and manage interactions efficiently.

---

## 🚀 Features

### 👨‍🎓 Student Features

* Signup/Login as Student
* Search teachers by subject or location
* View teacher profiles
* Send messages to teachers
* Give reviews & feedback
* Track interactions

### 👩‍🏫 Teacher Features

* Signup/Login as Teacher
* Create and manage profile
* Receive messages from students
* Reply to student queries
* View reviews & ratings
